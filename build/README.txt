Um den Server der WebGL-Anwendung 5 in einem Boot zu starten, wird folgende Software benötigt:

NodeJs

Ist NodeJs installiert, bitte folgenden Befehl in der Kommandozeile ausführen

npm install serve

Alternativ können Sie die Datei setup.bat ausführen.




Um den Server zu starten öffnen Sie nun die Datei RUN.bat
